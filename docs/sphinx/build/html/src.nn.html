src.nn module¶

This module provides a CNN auto-encoder. The model can be adjusted, such that 0-3 pooling operations will be performed.

class src.nn.ConvNet(c_in=1, c_out=1, img_h=64, img_w=64, act='LReLU', n_pooling=0)[source]¶

Bases: torch.nn.modules.module.Module

This class defines the auto-encoder model. Up to 3 pooling operations can be performed.

Note

Increasing the number of pooling operations will increase the depth of encoder and decoder networks.
