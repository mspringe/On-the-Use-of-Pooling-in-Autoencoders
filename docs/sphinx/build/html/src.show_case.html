src.show_case module¶

This Module provides a script, that saves the models outputs for specific images, stored in 'show_cases/imgs'. Images will be written to 'show_cases/predictions'.

src.show_case.write_imgs(model, dset, device, s_batch=1, model_name='model')[source]¶

Writing collages to 'show_cases/predictions'. A collage feature the original image the L channel and the predicted image in that order from left to right.

Parameters

  • model -- model to be used

  • dset -- data set, containing images in 'show_cases/imgs'

  • device -- GPU device

  • s_batch -- size of batch

  • model_name -- name of model
