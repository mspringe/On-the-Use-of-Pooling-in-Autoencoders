src package¶

This project trains and evaluates autoencoders for image colorization.

  • The general autoencoder is defined in :mod:'src.nn'

  • src.training provides a script and methods for training

  • src.eval provides a script and methods for evaluation

  • src.data provides a fast data loader for small data sets

  • src.show_case provides a script and methods to display selected images stored in "show_cases"

  • src.os_utils provides a helper method for cross platform path sanity
